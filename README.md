# Peegram Network Analysis Tool

An engineered, terminal-based network utility designed for high-speed active port discovery and passive live packet interception. Built as a modular CLI application for the pOS infrastructure framework.

## Features
* **Active Probe Matrix:** Concurrently maps open TCP ports across target systems using optimized thread pooling.
* **Passive Traffic Monitor:** Hooks into network interfaces to stream real-time Layer-3 and Layer-4 packet flows.
* **Global CLI Binding:** Installs natively into the host system context for global terminal execution.
